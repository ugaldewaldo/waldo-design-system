# tasks/

Lightweight, versioned to-do list. One `.md` file per task (kebab-case name).
Used instead of background-task chips — nothing runs automatically; these are
just notes you read when you want.

**Format per file:**

```md
# <Short title>

- **What:** one line
- **Where:** file / area
- **Why:** reason
- **Status:** todo | doing | done
```

Delete or set Status: done when handled.
