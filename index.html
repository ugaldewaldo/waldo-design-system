How to use

Rule of thumb: never reference primitives (zinc-200, green-500) directly from components. Always go through a semantic token. If you can't find one that fits your case, either add a new semantic token or rethink the use case.

Why translucent (alpha) tokens matter: when a layer sits over an image or another color (like the Monitor cards over the gradient hero), a solid color reads wrong. Use surface-frosted so the layer adapts to whatever's behind.

Migration Guide

Map every legacy style to its new token. Work top-to-bottom and use Figma's "Select all using this style" to swap in bulk.
