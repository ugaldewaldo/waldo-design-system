text-* track

UI, body, product surfaces

Tailwind standard scale 12→48. Default tracking -0.02em (-2%).

text-display-* track

Marketing heros + section headings

Custom scale 24 / 32 / 40 / 56 / 72 / 96. Default tracking -0.04em (-4%). Tailwind-native: utility class is text-display-md.
