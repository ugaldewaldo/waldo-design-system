Atomic components (waldo-ui/src/components/ui/*.tsx) are frozen unless explicitly approved for change. The demo adapts to the component — never the reverse.

Pre-commit guard

Install once per clone: bash tools/install-hooks.sh. On every commit it runs detect.js on staged code + lint-doctrine.js + lint-index.js + build-waldo-ds.sh --check + build-component-index.js --check + a YAML parse of both catalogs, and blocks the commit on any error. detect.js skips index.html (the showcase reference), tools/ (infra), and waldo-labs/ (prototype labs).
