# Email templates

Sending-ready HTML (dark-mode safe, mobile safe), built on the tokens in `DESIGN.md`:

- `base/` — shared skeleton: header, footer, type scale, buttons, link colors
- `welcome/` — first email after signup
- `newsletter/` — feature announcement / periodic newsletter with repeatable item blocks

Each template ships with a `copy-skeleton.md` (what text goes where, tone, lengths).
