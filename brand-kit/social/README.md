# Social post patterns

One folder per pattern, each containing:

- `spec.md` — anatomy: slots (headline / visual / meta), sizes, color family, rules
- `samples/` — 2–3 rendered examples (PNG) the generation skill uses as references

Master canvas formats (all patterns derive from these): feed portrait 1200×1350, square 1080×1080, landscape/OG 1200×630, video 16:9 1280×720, carousel slide 1080×1350.

Patterns: partnership · launch · stats · quote · event · case-study · article · carousel · teaser · video-card · recap · reactive
