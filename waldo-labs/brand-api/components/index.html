3 · Depth Pill + depthAlpha() encoding rule

The formula below IS the encoding rule — alpha 0.30 at 10% depth → 1.0 at 60%+, clamped. Background fixed at rgba(var(--highlight-rgb), 0.10); only the text alpha varies. A 70% pill must look heavier than a 20% pill — never flatten. Magnitude only, never status; never --chart-* here.
